## 코드 인라인  `` 백쿼트사용

블록 한개를 코드형식으로 보이게 할때 사용

예를들어 `a=123` , `b=1` 일때 `a+b = 124`



## 블록 생성

여러줄 ``` 엔터 블록생성 

```python
a=1
b=2
a-b=-1
print(a)
def my_func(n):
    return n
하단에 코딩언어 설정
```
